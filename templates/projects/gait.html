Human Gait Analysis using AI

What is Gait Analysis?

Gait analysis is the study of human locomotion, specifically how people walk or run. It is widely used in healthcare, sports, and security to detect abnormalities, enhance performance, or identify individuals based on their walking patterns.

About the Project

This Human Gait Analysis project uses Artificial Intelligence to track and analyze body movements in real time. Developed using OpenCV and Python, it leverages machine learning models to recognize gait patterns, predict anomalies, and identify individuals. The system can be used in rehabilitation, sports performance analysis, or even biometric authentication systems.

Features

  • Real-time gait tracking using OpenCV
  • Pose estimation and joint detection
  • Pattern classification and anomaly detection
  • Visual graph of movement dynamics
  • Trained on multiple public gait datasets
  • User-specific gait profiling
  • Portable and modular codebase

Technologies Used

  • Languages: Python
  • Libraries: OpenCV, Mediapipe, Scikit-learn
  • ML: KNN, Random Forest for classification
  • Visualization: Matplotlib, Seaborn
  • Development Tools: Jupyter Notebook, GitHub

Conclusion

The Human Gait AI project demonstrates how computer vision and machine learning can be combined for human motion analysis. It has applications in healthcare, security, sports, and biometric authentication. Through this project, I gained hands-on experience in pose estimation, real-time video processing, and ML model integration.
