CRM System

What is CRM?

CRM stands for Customer Relationship Management. It is a software system that helps businesses manage their interactions with current and potential customers. A CRM system allows teams to stay connected, streamline processes, and improve profitability by organizing customer information and communications in one place.

About the Project

A Customer Relationship Management (CRM) system helps businesses manage customer data, streamline processes, improve customer service, and boost sales. It allows teams to track interactions, manage leads, and organize customer-related tasks in one central place. This CRM System was developed during my internship at Ukshati Technologies to enhance customer and project management efficiency.

Features

  • Add, edit, and view customer information
  • Import/export customers via CSV
  • Manage project assignments
  • Set follow-up reminders with web notifications
  • Track project expenses and status
  • Admin login for secure access
  • Dashboard for quick navigation and insights

Technologies Used

  • Frontend: Next.js, HTML, CSS, JavaScript
  • Backend: Node.js
  • Database: MySQL
  • Containerization: Docker
  • Project Management: Scrum, GitHub

My Role as Scrum Master

During the internship, I was appointed as the Scrum Master for the CRM team in the 4th week. I facilitated daily stand-ups, managed sprint planning and retrospectives, and ensured blockers were addressed quickly. I also coordinated with other module teams (such as Inventory, Quotation, etc.) for smooth integration and helped maintain GitHub repositories and documentation. This role greatly enhanced my leadership, communication, and Agile project management skills.

Conclusion

This CRM System helped streamline customer and project tracking during the internship. It improved internal team coordination and taught valuable skills in full-stack development, teamwork, and real-world problem-solving.
